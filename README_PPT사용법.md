# PPT 이미지 자동 삽입 스크립트 사용법

## 📋 개요
89개 업체의 이미지를 자동으로 PPT에 삽입하는 Python 스크립트입니다.

## 🔧 필수 요구사항
- Python 3.6 이상
- 필요한 라이브러리: python-pptx, Pillow

## 📦 설치 방법

```bash
pip install python-pptx pillow
```

## 📁 예상 디렉토리 구조

```
C:\Users\user\Downloads\서울\중구\
├── 온아1호\
│   └── 업체\
│       ├── 업체_001.jpg
│       ├── 업체_002.jpg
│       └── ...
├── 온아2호\
│   └── 업체\
│       ├── 업체_001.jpg
│       ├── 업체_002.jpg
│       └── ...
└── ... (총 89개 업체 폴더)
```

## 🚀 사용 방법

### 1. 스크립트 파일 열기
`ppt_image_inserter.py` 파일을 텍스트 에디터로 엽니다.

### 2. 경로 설정 수정
스크립트 하단의 `main()` 함수에서 다음 값들을 수정하세요:

```python
# Windows 경로 예시
BASE_IMAGE_DIR = r"C:\Users\user\Downloads\서울\중구"

# 템플릿 PPT 파일
TEMPLATE_PPT = "세신샵.pptx"

# 출력 PPT 파일
OUTPUT_PPT = "세신샵_완성본.pptx"
```

### 3. 스크립트 실행

```bash
python ppt_image_inserter.py
```

## 📊 스크립트 동작 방식

1. **업체 디렉토리 스캔**: 지정된 경로에서 모든 업체 폴더를 찾습니다.
2. **이미지 파일 수집**: 각 업체의 `업체/업체_*.jpg` 파일들을 수집합니다.
3. **슬라이드 생성**: 
   - 업체명 표지 슬라이드
   - 가격표 슬라이드 (첫 번째 이미지)
   - 인테리어 슬라이드들 (나머지 이미지)
4. **이미지 자동 배치**: 샘플 PPT의 레이아웃에 맞춰 이미지를 자동으로 배치합니다.
5. **PPT 저장**: 최종 완성된 PPT를 저장합니다.

## 🎨 이미지 크기 및 배치

- **자동 비율 조정**: 이미지의 원본 비율을 유지하면서 슬라이드에 맞게 자동 조정
- **중앙 정렬**: 모든 이미지는 슬라이드 중앙에 배치
- **최대 크기**: 가로 11인치, 세로 5.5인치 이내로 자동 조정

## 📝 주의사항

1. **경로 형식**: Windows 경로는 `r"C:\경로"` 형식으로 작성하거나, `C:/경로` 형식으로 슬래시를 사용하세요.
2. **파일명 패턴**: 이미지 파일명은 `업체_001.jpg`, `업체_002.jpg` 형식이어야 합니다.
3. **폴더 구조**: 각 업체 폴더 내에 `업체` 하위 폴더가 있어야 합니다.

## 🔍 문제 해결

### 업체 디렉토리를 찾을 수 없습니다
- 경로가 정확한지 확인하세요.
- 폴더 구조가 예상 구조와 일치하는지 확인하세요.

### 이미지 추가 실패
- 이미지 파일이 손상되지 않았는지 확인하세요.
- 파일 권한을 확인하세요.

### 메모리 부족
- 이미지가 너무 많은 경우, 배치 처리로 나누어 실행하세요.

## 💡 커스터마이징

### 슬라이드 제목 변경
`add_shop_to_ppt` 메서드에서 `slide_title` 변수를 수정하세요.
